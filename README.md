# TypeScript Problem Solving

This repository contains solutions to various TypeScript problem-solving exercises focused on strengthening core TypeScript concepts and object-oriented programming principles.

## Topics Covered

### Basic Types & Essential Operators
- **Primitive Types:** string, number, boolean, null, undefined, symbol
- **Non-Primitive Types:** Array, Tuple, and Object
- **Advanced Syntax:** Rest & Spread operators, Destructuring
- **Type Logic:** Union and Intersection types, Literal types, and Optional types
- **Control Flow:** Ternary operator, Nullish Coalescing (`??`), and Optional Chaining (`?.`)
- **Special Types:** nullable, unknown, and `never` type

### Advanced Type Manipulation
- **Type Aliases vs. Interfaces**
- **Type Assertions** (as syntax)
- **Generics:** Generic interfaces, Generic functions, and Generic constraints
- **Keyof Constraints:** Using `keyof` with generics for type safety
- **Enums:** Usage of Enums and the `as const` pattern (Literal Inference)
- **Conditional Types:** Creating types based on logical conditions
- **Mapped Types:** Transforming existing types into new ones
- **Utility Types:** Built-in TypeScript helpers (Partial, Pick, Omit, etc.)

### Object-Oriented Programming (OOP)
- **Class Foundations:** Classes, objects, and constructors
- **The 4 Pillars of OOP:**
    - **Inheritance:** Extending functionality across classes
    - **Polymorphism:** Method overriding and dynamic behavior
    - **Abstraction:** Implementing abstract classes and interfaces
    - **Encapsulation:** Protecting data and internal logic
- **Access Modifiers:** public, private, and protected
- **Type Guards:** Narrowing types using `typeof`, `in`, and `instanceof`
- **Getters & Setters:** Controlled access to class properties
- **Statics:** Using `static` for class-level properties and methods
